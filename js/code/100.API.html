API的概念 API（Application Programming Interface，应用程序编程接口）是一些预先定义的函数，目的是提供应用程序与开发人员基于某软件或硬件得以访问一组例程的能力，而又无需访问源码，或理解内部工作机制的细节。 ·任何开发语言都有自己的API ·API的特征输入和输出（I/O） 如： var max = Math.max(1,2,3,4,5,6); ·API的使用方法(console.log()) Web API的概念 浏览器提供的一套操作浏览器功能和页面元素的API（BOM和DOM） 此处的Web API特指浏览器提供的API（一组方法）.WebAPl在后面的课程中有其它含义 掌握常见浏览器提供的API的调用方式
